🚀 COMO TESTAR
═════════════════════════════════════════════════════════════════════

⭐ OPÇÃO 1: Python (Recomendado - Abre automaticamente)
   
   python run-welcome.py
   
   → Inicia servidor em http://localhost:8000/
   → Abre no navegador automaticamente!

─────────────────────────────────────────────────────────────────

OPÇÃO 2: VS Code Live Server
   
   1. Instale extensão "Live Server"
   2. Clique direito em "welcome-pro.html"
   3. "Open with Live Server"

─────────────────────────────────────────────────────────────────

OPÇÃO 3: Terminal Raw
   
   python -m http.server 8000
   # Abra: http://localhost:8000/welcome.html

─────────────────────────────────────────────────────────────────

OPÇÃO 4: HTTP Server
   
   npx http-server
   # Abra: http://localhost:8080/welcome.html
